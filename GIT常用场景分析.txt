配置到代码仓库的ssh权限（假设本地已经有能够直接使用的ssh-key）
    新建~/.ssh/config文件（在当前登录用户的家文件夹下找到.ssh文件夹，新建config文件，并根据以下内容进行配置）
        Host github.com
        HostName github.com
        User git
        Port 22
        IdentityFile ~/.ssh/windows_github
    配置完成之后测试是否有到github的权限
        ssh -T git@github.com
